## gen-cli

Generate project seed tools.

### Instal

    npm i gen-cli -g

### Usage

0. put your project seed in `templates`. eg: `express-ejs`.
1. run `gen express-ejs myapp`.
2. run `cd myapp && npm init`.
3. run `npm install` or `npm install xxx --save` or `npm install xxx --save-dev`.
